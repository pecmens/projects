## 部署步骤

1. 运行 `npm run build` 构建项目
2. 运行 `npm run export` 生成静态文件
3. 检查 `out` 目录中的静态文件
4. 部署到静态托管服务（如GitHub Pages、Netlify、Vercel等）

## GitHub Pages 部署说明

1. 构建项目：`npm run build && npm run export`
2. 将 `out` 目录中的内容推送到 `gh-pages` 分支
3. 在GitHub仓库设置中启用Pages，选择 `gh-pages` 分支
